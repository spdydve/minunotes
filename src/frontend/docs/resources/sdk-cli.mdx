# SDK and CLI

The SDK and CLI target the harness API and use the same `X-API-Key` permission model.

## SDK

```ts
import { NotesClient } from "@minunotes/sdk";

const notes = new NotesClient({
  baseUrl: "https://your-minunotes-host",
  apiKey: process.env.NOTES_API_KEY!,
});

const folders = await notes.folders.list();
const search = await notes.notes.search("project");
```

## CLI

```sh
export NOTES_API_URL="https://your-minunotes-host"
export NOTES_API_KEY="ntak_..."

notes folders list
notes notes create --folder <folder-id> --title "Hello" --content ./note.md
notes notes edit <note-id> --old "before" --new "after" --base-hash <hash>
notes search "query" --json
```
