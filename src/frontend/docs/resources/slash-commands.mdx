# Slash commands

Slash commands help insert common Markdown blocks without remembering every syntax detail.

## Open the menu

Type `/` at the start of an empty line.

Keep typing to filter commands, use arrow keys to move through options, and press Enter to insert.

## Common commands

- Heading 1
- Heading 2
- Heading 3
- Bulleted List
- Numbered List
- Task List
- Quote
- Code Block
- Table
- Divider
- Image
- Wiki Link

## Image command

Use the Image command to open the MinuNotes image picker. You can upload an image or insert an image URL.

## Wiki Link command

Use Wiki Link to insert a note link placeholder:

```md
[[]]
```

If text is selected, the command wraps that text as a wikilink:

```md
[[selected text]]
```

You can also type `[[` directly to open note suggestions.
