# OpenAPI

MinuNotes exposes a static OpenAPI 3.1 document for harness endpoints.

## Spec URLs

```txt
GET /openapi.json
GET /v1/harness/openapi.json
```

Both URLs return the same harness API specification.

## Use cases

Use OpenAPI when a platform imports REST tools from a schema, such as custom actions, hosted workflow tools, or non-MCP agent platforms.

## Auth scheme

The spec defines `ApiKeyAuth` using the `X-API-Key` header.

```http
X-API-Key: ntak_...
```

## Scope

The spec describes `/v1/harness/*` endpoints. It does not duplicate hosted MCP protocol calls. MCP clients should use `/mcp` instead.
